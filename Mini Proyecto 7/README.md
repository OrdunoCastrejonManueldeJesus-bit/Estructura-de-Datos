Esta aplicación de consola en C++ implementa un 
Árbol Binario de Búsqueda (BST), permitiendo al usuario 
construir y manipular la estructura de datos mediante comandos interactivos.

-Descripción General-
El programa ofrece una interfaz de línea de comandos para realizar las operaciones 
fundamentales en un BST, incluyendo la gestión de nodos, la visualización de recorridos 
y la obtención de propiedades del árbol (como la altura y el tamaño). También permite la 
persistencia de datos mediante la carga y guardado de los elementos del árbol en archivos.

Funcionalidades Principales:
*Gestión de Nodos: Inserción, búsqueda (mostrando la ruta) y eliminación de elementos.
*Recorridos: Muestra los recorridos Inorder, Preorder y Postorder. * Propiedades del Árbol: Cálculo de la altura y el número de nodos (tamaño).
*Archivos: Exportación del recorrido Inorder a un archivo, carga de elementos desde un archivo y guardado del estado actual del árbol.
